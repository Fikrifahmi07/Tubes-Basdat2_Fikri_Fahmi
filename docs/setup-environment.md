# Setup Environment

## Requirement
- XAMPP 8.2+
- PHP 8.2+
- MariaDB / MySQL
- CodeIgniter 4

## PHP Extension Required
- intl
- mysqli
- mbstring

## Langkah Setup
1. Install / Upgrade XAMPP ke versi 8.2+
2. Aktifkan extension intl pada php.ini
3. Restart Apache
4. Letakkan project di folder htdocs
5. Jalankan project melalui browser
